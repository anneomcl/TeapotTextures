PLATFORM: Windows 7
Build Environment: Visual Studio 2013
Language: C++
External Libraries: glew, glut, gl, glm (GL Mathematics)

Build Instructions:

Open the solution in this folder in Visual Studio and press "play".
Make sure glew, glut, gl and glm are installed on your system properly.
Press 'e' to get the environment mapping and 't' to get the texture mapping.

I was not able to get multitexturing to work but you can continously toggle
between these two options. 

There is a .exe file in the "Debug" folder within this current folder.

The Youtube link is in a text file in this folder.
